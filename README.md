# Edymax Networks & Systems

## About Edymax

Edymax Networks & Systems is a comprehensive e-commerce platform built with Laravel that specializes in networking products and systems. We offer a wide range of technology solutions including:

### Core Product Categories
- **Networking Equipment**: Routers, switches, firewalls, access points, and network cables
- **Security Systems**: CCTV cameras, surveillance systems, access control, and security software
- **Computer Systems**: Laptops, desktops, workstations, and servers
- **Phone & Computer Accessories**: Keyboards, mice, monitors, chargers, and protective cases
- **Software Solutions**: Operating systems, productivity software, security software, and specialized applications
- **VoIP Telephony**: IP phones, PBX systems, VoIP gateways, and telecommunication solutions
- **Additional Products**: Printers, scanners, storage devices, and various IT peripherals
